🎉 Sofia - Hola Mundo

¿Qué es este módulo?

Es nuestro primer módulo personalizado que demuestra:

  • Creación de un modelo simple
  • Vistas de árbol y formulario
  • Acciones personalizadas
  • Integración con menús

Características

  • Crea registros con un nombre y mensaje
  • Botón "Saludar" que imprime en logs
  • Vista de listado y detalle
  • Fecha de creación automática

Cómo usar

  1. Instalar el módulo
  2. Ir a "Sofia Decorations" → "Hola Mundo"
  3. Crear un nuevo registro
  4. Hacer clic en "Saludar"
