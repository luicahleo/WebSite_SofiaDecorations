🎨 Sofia Decorations - Módulo Base

Descripción

Módulo base que contiene las configuraciones y funcionalidades comunes para toda la aplicación Sofia Decorations.

Características

  • Configuración centralizada de Sofia Decorations
  • Menús y acceso rápido
  • Base para módulos personalizados adicionales

Requisitos

  • Odoo 17.0
  • Módulos base: sale, stock, account

Instalación

1. Ir a Apps en el menú principal

2. Buscar "Sofia Decorations - Base"

3. Hacer clic en "Instalar"
